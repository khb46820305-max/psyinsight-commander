# 자동화 작동 방식 설명

## 🤔 현재 작동 방식

### 현재 상태
- **Streamlit 앱**: 웹 브라우저에서 접속해서 사용
- **자동화 스크립트**: `collect_news.py`, `collect_papers.py`
- **문제점**: 스크립트를 수동으로 실행하거나 작업 스케줄러에 등록해야 함

---

## 💡 원하는 작동 방식

**"컴퓨터만 켜져있으면 자동으로 작동"**

### 해결 방법

#### 방법 1: Windows 작업 스케줄러 (현재 방식)
- ✅ PC만 켜져있으면 자동 실행
- ✅ 프로그램 실행 불필요
- ⚠️ PC가 꺼지면 중단

#### 방법 2: 백그라운드 서비스로 실행 (개선)
- PC 부팅 시 자동 시작
- 항상 백그라운드에서 실행
- 더 안정적

#### 방법 3: 클라우드 배포 (완전 자동화)
- Streamlit Cloud (무료)
- 24/7 자동 실행
- PC 꺼져도 작동

---

## 📱 앱으로 만들기

### 모바일 앱의 한계
- ❌ 백그라운드 실행 제한 (배터리 절약)
- ❌ 정기적 자동 실행 어려움
- ❌ 서버 역할 불가

### 대안: 클라우드 배포
- ✅ 24/7 자동 실행
- ✅ PC 꺼져도 작동
- ✅ 무료 (Streamlit Cloud)

---

## 🌐 도메인 비용

### 무료 옵션
- **Streamlit Cloud**: 무료, 자동 도메인 제공
  - 예: `yourname-psyinsight-commander.streamlit.app`
- **커스텀 도메인**: 유료 (연간 $10-20)

### 유료 옵션
- 커스텀 도메인 연결: 연간 $10-20
- 전용 서버: 월 $5-20

---

## 🎯 추천 솔루션

### 완전 자동화를 원한다면:

1. **Streamlit Cloud에 배포** (무료)
   - 24/7 자동 실행
   - PC 꺼져도 작동
   - 자동 도메인 제공

2. **GitHub Actions로 자동 수집** (무료)
   - 매일 자동으로 스크립트 실행
   - 수집 후 이메일 발송
   - 완전 자동화

3. **로컬 PC 사용 시**
   - Windows 작업 스케줄러 설정
   - PC만 켜져있으면 자동 실행
