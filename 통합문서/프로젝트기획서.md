## 프로젝트 개요

**PsyInsight Commander**는 심리학 전문가를 위한 통합 대시보드로, 심리 관련 뉴스와 논문을 자동으로 수집, 분석하고 다양한 형태의 콘텐츠로 재생산하는 플랫폼입니다.

### 핵심 기능
1. **뉴스 수집 및 분석**: 한국/미국 심리 관련 뉴스 자동 수집 및 AI 분석
2. **논문 수집 및 해석**: arXiv/PubMed에서 논문 수집 및 Abstract 해석
3. **콘텐츠 재생산**: 블로그, 릴스 대본, 게시글, 논문 아이디어 생성
4. **자동화**: 매일 자동 수집 및 이메일 알림

### 기술 스택
- **Frontend**: Streamlit
- **Backend**: Python
- **Database**: SQLite
- **AI**: Google Gemini API
- **Deployment**: Streamlit Cloud
