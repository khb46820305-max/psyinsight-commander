## 다른 PC로 이동하기

### 필수 사항
1. **Python 설치**: Python 3.8 이상
2. **의존성 설치**: `pip install -r requirements.txt`
3. **환경 변수 설정**: `.env` 파일 수동 생성
4. **데이터베이스 초기화**: `python -m modules.database`

### 주의사항
- `.env` 파일은 Git에 포함되지 않음 (보안)
- 데이터베이스 파일(`data/psyinsight.db`)도 함께 복사 필요

## 프로젝트 맥락

**새로운 AI 세션 시작 시 필수 읽기!**

이 프로젝트는 심리학 전문가를 위한 통합 대시보드입니다.

### 핵심 기능
- 뉴스/논문 자동 수집
- AI 기반 분석
- 콘텐츠 재생산
- 완전 자동화

### 기술 스택
- Streamlit (Frontend)
- Python (Backend)
- SQLite (Database)
- Gemini API (AI)

## 질문과 답변

### Q: 도메인 비용이 들나요?
A: Streamlit Cloud는 무료로 사용 가능합니다.

### Q: 자동 수집은 어떻게 되나요?
A: Windows Task Scheduler 또는 GitHub Actions를 사용합니다.

### Q: 다른 PC로 이동하려면?
A: 폴더 복사만으로는 안 되고, Python 설치 및 의존성 설치가 필요합니다.

## 토큰 관리

### GitHub Personal Access Token
- 생성: Settings > Developer settings > Personal access tokens
- 권한: `repo` (전체)
- 보관: 안전한 곳에 저장 (한 번만 표시됨)

### Gemini API Key
- 생성: https://makersuite.google.com/app/apikey
- 보관: `.env` 파일 또는 Streamlit Secrets
