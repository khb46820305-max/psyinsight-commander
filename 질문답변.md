## Q3. 원하는 작동 방식

**"컴퓨터만 켜져있으면 자동으로 작동"**

### 해결 방법

#### 방법 1: Windows 작업 스케줄러 (로컬 PC)
1. 작업 스케줄러에 등록
2. PC만 켜져있으면 자동 실행
3. 프로그램 실행 불필요
4. 매일 새벽 2시 자동 수집
5. 수집 완료 후 자동 이메일 발송
6. 데이터베이스에 자동 저장
7. 필요할 때 웹에서 확인

**설정 방법:**
- `Win + R` → `taskschd.msc`
- "기본 작업 만들기"
- 프로그램: `python`
- 인수: `C:\EX\collect_news.py`
- 트리거: 매일 새벽 2시
- **"사용자가 로그온할 때만 실행" 체크 해제**

#### 방법 2: 클라우드 배포 (완전 자동화, 추천!)
- Streamlit Cloud에 배포
- GitHub Actions로 자동 수집
- PC 꺼져도 작동
- 24/7 자동 실행

## 현재 작동 방식

### 현재 상태
- Streamlit 앱: 웹 브라우저에서 접속
- 수동 수집: 버튼 클릭으로 수집
- 자동화 스크립트: 수동 실행 필요

### 완전 자동화 후
1. **Windows 작업 스케줄러 설정**
   - PC만 켜져있으면 자동 실행
   - 매일 새벽 2시 자동 수집
   - 프로그램 실행 불필요

2. **자동 작동 흐름**
   ```
   매일 새벽 2시 (자동)
     ↓
   collect_news.py 실행 (작업 스케줄러)
     ↓
   뉴스 수집 → AI 분석 → DB 저장
     ↓
   이메일 자동 발송
     ↓
   데이터베이스에 누적 저장
   ```

3. **사용자가 할 일**
   - 필요할 때 웹 브라우저에서 접속
   - 수집된 데이터 확인
   - 검색/필터로 원하는 내용 찾기
