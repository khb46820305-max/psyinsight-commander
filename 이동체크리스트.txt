다른 PC로 프로젝트 이동 체크리스트
=====================================

[현재 PC에서]
□ 프로젝트 폴더 전체 복사 (C:\EX\)
□ .env 파일 포함 확인
□ data/psyinsight.db 복사 여부 결정 (기존 데이터 필요 시)

[새 PC에서]
□ Python 3.9 이상 설치 확인
□ 프로젝트 폴더 이동
□ pip install -r requirements.txt 실행
□ .env 파일 확인 (없으면 생성)
□ python -m modules.database 실행
□ streamlit run app.py 테스트

주의사항:
- .env 파일은 반드시 복사해야 함 (API 키 포함)
- 라이브러리는 새 PC에서 재설치 필수
- Python 버전 확인 필수
