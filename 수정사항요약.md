# 수정 사항 요약

## ✅ 완료된 수정사항

### 1. 뉴스 수집 키워드 개선 ✅

**문제:**
- "심리", "상담"만으로 검색하면 관련 없는 뉴스도 수집됨 (건설근로자공제회, 디지털 상담창구 등)

**해결:**
- 더 구체적인 키워드 사용:
  - 한국: "정신건강", "심리건강", "마음건강", "심리상담", "심리학이론", "심리학", "정신건강증진", "우울증", "불안장애", "트라우마", "상담심리", "임상심리"
  - 미국: "psychology", "mental health", "counseling psychology", "clinical psychology", "depression", "anxiety", "trauma"
- 검색 쿼리에 제외 키워드 추가: `-법률 -법무 -디지털 -IT -건설 -공제회`
- 제목 기반 관련성 필터링: 심리/정신건강 관련 키워드가 제목에 없으면 스킵

### 2. 뉴스 요약 생성 실패 처리 개선 ✅

**문제:**
- "요약 생성에 실패했습니다." 메시지가 그대로 표시됨

**해결:**
- 요약 생성 실패 시 텍스트의 첫 부분을 요약으로 사용
- 최소 길이 체크 추가
- 더 나은 fallback 메커니즘

### 3. 논문 수집 피드백 추가 ✅

**문제:**
- 논문 수집 버튼 클릭 시 작동 여부가 보이지 않음

**해결:**
- Spinner 표시 (이미 있었지만 개선)
- 성공/실패 메시지 개선
- 수집된 논문이 없을 때 경고 메시지 표시
- 오류 발생 시 상세한 traceback 표시

### 4. 콘텐츠 팩토리 기능 구현 ✅

**문제:**
- 템플릿 선택 후 콘텐츠 생성이 되지 않음

**해결:**
- 콘텐츠 생성 로직 완전 구현
- 4가지 템플릿 지원:
  - 블로그 포스트
  - 릴스 대본
  - 게시글
  - 논문 아이디어
- 생성된 콘텐츠 표시 및 복사 기능
- 에러 처리 개선
