# PsyInsight Commander 최종 요약 보고서

## 🎯 프로젝트 개요

**심리학 전문가를 위한 자동화 대시보드 웹 애플리케이션**

---

## ✅ 핵심 기능 (3개 Tab)

### 1️⃣ Tab 1: 사이콜로지 트랜드 레이더
- **뉴스 자동 수집**: 미국/한국 심리 관련 뉴스
- **AI 요약 및 평가**: 3줄 요약 + 전문성 점수 (1~5점)
- **검색/필터**: 키워드, 해시태그, 평점별 검색

### 2️⃣ Tab 2: 아카데믹 아카이브
- **논문 자동 수집**: arXiv, PubMed
- **AI 요약**: 연구 목적/방법/결과/시사점
- **카테고리 분류**: 심리학/상담/교정심리/범죄심리

### 3️⃣ Tab 3: 콘텐츠 팩토리
- **콘텐츠 재생산**: 선택한 뉴스/논문을 4가지 형태로 변환
  - 블로그 포스트
  - 릴스 대본
  - 게시글
  - 논문 아이디어

---

## 🔄 완전 자동화

### 자동 수집
- ✅ **앱 접속 불필요**: 별도 스크립트로 독립 실행
- ✅ **매일 자동**: 뉴스는 매일, 논문은 매주
- ✅ **Windows 작업 스케줄러**: PC만 켜져있으면 자동 실행

### 이메일 자동 발송
- ✅ **수집 완료 후 자동 발송**: 헤드라인 요약 형식
- ✅ **Gmail SMTP**: 무료 사용
- ✅ **설정 간단**: 환경 변수만 설정하면 완료

---

## 💰 비용

**완전 무료**
- Gemini API: 무료 티어
- Streamlit Cloud: 무료 배포
- Gmail SMTP: 무료
- GitHub Actions: 무료 (자동화)

---

## ⏱️ 개발 시간

- **MVP**: 30-40시간
- **전체**: 60-90시간

---

## 🛠️ 기술 스택

- **Frontend**: Streamlit (Python)
- **Backend**: Python + SQLite
- **AI**: Google Gemini API
- **자동화**: Python 스크립트 + 작업 스케줄러

---

## 📧 작동 방식

```
매일 새벽 2시 (자동)
  ↓
뉴스 수집 → AI 분석 → DB 저장
  ↓
이메일 자동 발송 (헤드라인 요약)
  ↓
당신의 이메일로 도착! 📧
```

---

## 🎯 핵심 가치

1. **완전 자동화**: 앱 접속 없이도 매일 자동 수집
2. **이메일 알림**: 수집된 내용을 이메일로 자동 발송
3. **비용 효율**: 무료 티어만 사용
4. **AI 기반**: 자동 요약 및 평가
