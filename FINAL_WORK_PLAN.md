# PsyInsight Commander 최종 작업계획서

## 📋 프로젝트 목표
심리학 전문가를 위한 통합 대시보드 웹 애플리케이션 개발
- 뉴스, 논문 수집/분석 (Tab 1, 2)
- AI 기반 요약 및 평가
- 콘텐츠 재생산 기능 (Tab 3)
- 완전 자동화 + 이메일 발송

---

## 📊 전체 작업 개요

| Phase | 내용 | 예상 시간 | 우선순위 |
|-------|------|----------|---------|
| Phase 1 | MVP 개발 | 30-40시간 | 최고 |
| Phase 2 | 확장 기능 | 20-30시간 | 높음 |
| Phase 3 | 고도화 | 10-20시간 | 중간 |
| **총계** | **전체 프로젝트** | **60-90시간** | - |
