# 자동화 작동 방식 상세 설명

## 🎯 원하는 작동 방식

**"컴퓨터만 켜져있으면 자동으로 작동"**
- 프로그램 실행 불필요
- 매일 자동 수집
- 자동 이메일 발송
- 데이터베이스에 누적 저장
- 필요할 때만 웹에서 확인

---

## ✅ 해결 방법: Windows 작업 스케줄러

### 작동 원리

1. **작업 스케줄러에 등록**
   - `collect_news.py` 스크립트를 매일 새벽 2시에 자동 실행
   - PC만 켜져있으면 자동 실행
   - 프로그램 실행 불필요

2. **자동 작동 흐름**
   ```
   매일 새벽 2시 (자동, 작업 스케줄러)
     ↓
   collect_news.py 실행
     ↓
   뉴스 수집 (Google News RSS)
     ↓
   AI 분석 (요약, 평가, 키워드)
     ↓
   데이터베이스에 저장 (누적)
     ↓
   이메일 자동 발송 (헤드라인 요약)
     ↓
   완료!
   ```

3. **사용자가 할 일**
   - 아무것도 안 해도 됨!
   - 필요할 때만 웹 브라우저에서 접속
   - `streamlit run app.py` 실행
   - 수집된 데이터 확인

## 📱 앱 vs 웹사이트

### 모바일 앱의 문제
- ❌ 백그라운드 실행 제한
- ❌ 정기적 자동 실행 어려움
- ❌ 서버 역할 불가

### 웹사이트의 장점
- ✅ 클라우드에 배포하면 24/7 자동 실행
- ✅ PC 꺼져도 작동
- ✅ 폰에서도 접속 가능 (웹 브라우저로)

## 🔧 완전 자동화 설정 방법

### 방법 1: 로컬 PC + 작업 스케줄러

1. 작업 스케줄러 열기
   - `Win + R` → `taskschd.msc`

2. 기본 작업 만들기
   - 이름: "PsyInsight 뉴스 수집"
   - 트리거: 매일, 새벽 2시
   - 작업: 프로그램 시작
   - 프로그램: `python`
   - 인수: `C:\EX\collect_news.py`
   - 시작 위치: `C:\EX`
   - **"사용자가 로그온할 때만 실행" 체크 해제**
   - **"가장 높은 수준의 권한으로 실행" 체크**

3. 완료!
   - PC만 켜져있으면 매일 자동 실행
   - 프로그램 실행 불필요

### 방법 2: 클라우드 배포 (완전 자동화)

1. Streamlit Cloud에 배포
2. GitHub Actions 설정
3. PC 꺼져도 작동
4. 24/7 자동 실행
