# PsyInsight Commander - 프로젝트 맥락 문서

> **이 문서는 AI가 프로젝트의 전체 맥락을 빠르게 이해하기 위한 문서입니다.**

---

## 🎯 프로젝트 개요

**PsyInsight Commander**: 심리학 전문가를 위한 통합 대시보드 웹 애플리케이션

### 핵심 목표
- 심리학 관련 뉴스와 논문을 자동으로 수집, 분석
- AI 기반 요약 및 전문성 평가
- 다양한 형태의 콘텐츠로 재생산
- 완전 자동화 + 이메일 알림

---

## 🏗️ 프로젝트 구조

```
C:\EX\
├── app.py                    # 메인 Streamlit 앱 (3개 Tab)
├── collect_news.py          # 뉴스 수집 자동화 스크립트
├── collect_papers.py        # 논문 수집 자동화 스크립트
├── requirements.txt         # Python 라이브러리 목록
├── .env                     # 환경 변수 (API 키, 이메일 설정)
├── modules/
│   ├── database.py          # SQLite 데이터베이스 관리
│   ├── ai_engine.py         # Gemini API 연동 (요약, 평가, 키워드)
│   ├── news_collector.py   # 뉴스 수집 (Google News RSS)
│   ├── paper_collector.py   # 논문 수집 (arXiv, PubMed)
│   └── email_sender.py     # 이메일 발송 (Gmail SMTP)
└── data/
    └── psyinsight.db       # SQLite 데이터베이스
```

## ✅ 완료된 기능

### Tab 1: 사이콜로지 트랜드 레이더
- ✅ 뉴스 수집 (한국/미국, Google News RSS)
- ✅ AI 요약 및 전문성 평가 (1~5점)
- ✅ 키워드 추출
- ✅ 카드 형태 표시, 페이지네이션

### Tab 2: 아카데믹 아카이브
- ✅ 논문 수집 (arXiv, PubMed)
- ✅ AI 요약 (목적/방법/결과/시사점)
- ✅ 논문 목록 표시, 페이지네이션

### Tab 3: 콘텐츠 팩토리
- ✅ 뉴스/논문 선택
- ✅ 4가지 템플릿 (블로그, 릴스, 게시글, 논문아이디어)
- ✅ AI 콘텐츠 생성

### 자동화
- ✅ 독립 실행 스크립트 (`collect_news.py`, `collect_papers.py`)
- ✅ 이메일 자동 발송
- ✅ Windows 작업 스케줄러 설정 가이드

## 🔧 기술 스택

- **Frontend**: Streamlit (Python)
- **Backend**: Python 3.9+
- **Database**: SQLite
- **AI**: Google Gemini API (gemini-1.5-flash)
- **이메일**: Gmail SMTP
- **데이터 수집**: RSS Feed, arXiv API, PubMed API
