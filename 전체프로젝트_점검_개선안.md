# PsyInsight Commander 전체 프로젝트 점검 및 개선안

## 📊 현재 프로젝트 구조

### 메뉴 구성
1. **📰 트랜드 레이더**: 심리 관련 뉴스 수집 및 분석
2. **📚 아카이브**: 심리 관련 논문 수집 및 해석
3. **✨ 팩토리**: 콘텐츠 재생산 (블로그, 릴스, 게시글, 논문 아이디어)
4. **📈 경제 흐름 파악**: 경제 뉴스 수집 및 분석
5. **🗑️ 수집 내용 관리**: 수집된 뉴스/논문 삭제
6. **🧪 테스트**: 빠른 테스트 수집
7. **⚙️ 설정**: 환경 설정
8. **🗄️ 초기화**: 데이터베이스 초기화

---

## 🔍 기능별 점검 및 문제점

### 1. 📰 트랜드 레이더 (뉴스 수집)

#### 현재 상태
- ✅ 한국/미국 뉴스 수집
- ✅ AI 요약 및 평가
- ✅ 키워드 추출
- ✅ 검색/필터 기능
- ✅ 페이지네이션

#### 문제점
1. **"트렌드" 파악 어려움**
   - 개별 뉴스만 나열되어 시간에 따른 변화를 파악하기 어려움
   - 예: "이번 주에 '트라우마' 관련 뉴스가 증가했는가?" 파악 불가

2. **중요도 표시 부족**
   - 모든 뉴스가 동일한 중요도로 표시됨
   - 사용자가 중요한 뉴스를 찾기 위해 모든 뉴스를 읽어야 함

3. **시각화 부족**
   - 텍스트 기반만 제공
   - 키워드 트렌드, 카테고리별 분포 등을 시각적으로 보기 어려움

#### 개선안
1. **트렌드 대시보드 추가**
   - 최근 7일간 키워드별 뉴스 개수 변화 그래프
   - 카테고리별 분포 파이 차트
   - 국가별(한국/미국) 비교 차트

2. **주요 이슈 하이라이트**
   - 키워드 빈도수 기반 자동 선별
   - 최대 5개 주요 이슈 상단 표시

3. **스마트 필터링**
   - "인기 키워드" 클릭으로 해당 키워드 뉴스만 필터링
   - "트렌드 상승" 키워드 자동 표시

---

### 2. 📚 아카이브 (논문 수집)

#### 현재 상태
- ✅ arXiv/PubMed 논문 수집
- ✅ 저명 학술지 필터링
- ✅ Abstract 해석 (외국 논문)
- ✅ 검색/필터 기능
- ✅ 페이지네이션

#### 문제점
1. **논문 간 연관성 파악 어려움**
   - 개별 논문만 표시되어 관련 논문들을 그룹화하기 어려움
   - 예: "트라우마 치료" 관련 논문들이 여러 개 있어도 연결되지 않음

2. **연구 동향 파악 어려움**
   - 시간에 따른 연구 주제 변화를 파악하기 어려움
   - 예: "최근 1년간 어떤 주제의 논문이 많이 발표되었는가?"

3. **저널 신뢰도 정보 부족**
   - 저명 학술지 여부는 표시되지만, 구체적인 영향력 지수(IF) 등은 없음

#### 개선안
1. **연구 동향 분석**
   - 최근 6개월/1년간 주제별 논문 개수 변화 그래프
   - 키워드 클라우드로 주요 연구 주제 시각화

2. **논문 그룹화**
   - 유사 키워드를 가진 논문들을 자동으로 그룹화
   - "관련 논문" 섹션 추가

3. **저널 정보 강화**
   - 저널명 클릭 시 해당 저널의 최근 논문 목록 표시
   - 저널별 논문 개수 통계

---

### 3. ✨ 팩토리 (콘텐츠 생성)

#### 현재 상태
- ✅ 뉴스/논문 선택
- ✅ 4가지 콘텐츠 형식 생성 (블로그, 릴스, 게시글, 논문 아이디어)
- ✅ 진행률 표시

#### 문제점
1. **생성된 콘텐츠 관리 부족**
   - 생성된 콘텐츠를 저장하거나 다시 볼 수 없음
   - 예: 어제 생성한 블로그 포스트를 다시 보고 싶어도 불가능

2. **콘텐츠 품질 검증 어려움**
   - 생성된 콘텐츠가 적절한지 확인하기 어려움
   - 수정이나 재생성 기능 없음

3. **템플릿 커스터마이징 불가**
   - 사용자가 원하는 형식으로 수정하기 어려움
   - 예: 블로그 포스트 길이, 톤 등을 조절 불가

#### 개선안
1. **콘텐츠 저장소 추가**
   - 생성된 콘텐츠를 데이터베이스에 저장
   - "내 콘텐츠" 메뉴에서 저장된 콘텐츠 확인/수정/삭제

2. **콘텐츠 편집 기능**
   - 생성된 콘텐츠를 직접 수정 가능
   - "재생성" 버튼으로 일부만 다시 생성

3. **템플릿 설정**
   - 블로그 포스트 길이, 톤, 형식 등 커스터마이징
   - 사용자별 프리셋 저장

4. **콘텐츠 내보내기**
   - 마크다운, 텍스트, PDF 등으로 다운로드
   - 소셜 미디어에 바로 공유 가능한 형식

### 5. 🗑️ 수집 내용 관리

#### 현재 상태
- ✅ 뉴스/논문 선택 삭제
- ✅ 일괄 삭제 기능

#### 문제점
1. **삭제 전 확인 부족**
   - 실수로 중요한 뉴스/논문을 삭제할 위험
   - 삭제 전 미리보기 없음

2. **삭제 이력 관리 없음**
   - 삭제한 내용을 복구할 수 없음
   - 삭제 이력 확인 불가

#### 개선안
1. **삭제 전 확인 다이얼로그**
   - 삭제 전 선택한 항목 미리보기
   - "정말 삭제하시겠습니까?" 확인

2. **휴지통 기능** (선택)
   - 삭제된 항목을 30일간 보관
   - 복구 기능 제공

### 6. 🧪 테스트

#### 문제점

#### 개선안

   - "정상 작동" / "문제 발견" 자동 판단

## 🎯 전체 프로젝트 공통 문제점

### 1. **데이터 시각화 부족**
- 모든 탭이 텍스트 기반
- 그래프, 차트 등 시각적 요소 부족
- "트렌드", "흐름" 파악이 어려움

### 2. **사용자 경험(UX) 개선 필요**
- 각 탭이 독립적으로 작동하여 연관성 파악 어려움
- 예: "이 뉴스와 관련된 논문은?" 같은 연결 기능 없음
- 첫 사용자가 이해하기 어려운 구조

### 3. **데이터 활용도 낮음**
- 수집된 데이터를 단순 나열만 함
- 통계, 분석, 인사이트 도출 기능 부족
- 예: "이번 달 가장 많이 언급된 심리학 주제는?"

### 4. **콘텐츠 관리 기능 부족**
- 생성된 콘텐츠를 저장/관리할 수 없음
- 수집한 뉴스/논문을 북마크하거나 태그할 수 없음

### 5. **자동화 기능 가시성 부족**
- 자동 수집이 작동하는지 확인하기 어려움
- 마지막 수집 시간, 다음 수집 예정 시간 등 표시 없음

## 🚀 종합 개선안

### Phase 1: 핵심 기능 강화 (우선순위 높음)

#### 1.1 대시보드 통합
```
메인 대시보드 페이지 추가
├─ 오늘의 주요 이슈 (뉴스 + 논문 통합)
├─ 최근 7일 트렌드 그래프
├─ 키워드 클라우드
└─ 빠른 액션 버튼 (수집, 생성 등)
```

#### 1.2 트렌드 분석 기능
- 각 탭에 트렌드 그래프 추가
- 키워드별, 카테고리별 시간에 따른 변화 시각화
- "트렌드 상승" 키워드 자동 표시

#### 1.3 콘텐츠 저장소
- 생성된 콘텐츠 저장 기능
- 북마크 기능 (뉴스/논문)
- 태그 시스템

### Phase 2: 사용자 경험 개선

#### 2.1 연관성 연결
- 뉴스와 관련 논문 자동 연결
- "이 뉴스와 관련된 논문 보기" 기능
- "이 논문과 관련된 뉴스 보기" 기능

#### 2.2 스마트 추천
- 사용자가 관심 있어 할 만한 뉴스/논문 추천
- 읽은 항목 기반 개인화

#### 2.3 검색 기능 강화
- 통합 검색 (뉴스 + 논문 + 경제 뉴스)
- 고급 검색 (날짜 범위, 키워드 조합 등)

### Phase 3: 고급 기능 (선택)

#### 3.1 리포트 생성
- 주간/월간 리포트 자동 생성
- 주요 이슈 요약, 통계 등 포함

#### 3.2 알림 시스템
- 중요 뉴스/논문 알림
- 키워드 기반 알림 설정

#### 3.3 협업 기능
- 콘텐츠 공유
- 팀원과 협업 (선택)

## 💡 핵심 개선 포인트 요약

### 문제점
1. **"트렌드" 파악 불가**: 개별 항목만 나열
2. **시각화 부족**: 텍스트 기반만 제공
3. **데이터 활용도 낮음**: 수집만 하고 분석 부족
4. **콘텐츠 관리 없음**: 생성한 콘텐츠 저장 불가
5. **연관성 부족**: 뉴스-논문 간 연결 없음

### 개선안
1. **대시보드 추가**: 각 탭에 트렌드 그래프, 통계 추가
2. **시각화 강화**: 그래프, 차트, 클라우드 등 추가
3. **분석 기능**: 키워드 트렌드, 연구 동향 분석
4. **저장소 구축**: 콘텐츠 저장, 북마크, 태그
5. **연관성 연결**: 뉴스-논문 자동 연결

---

이 개선안을 통해 **"PsyInsight Commander"**가 단순한 수집 도구가 아닌, **진정한 인사이트 도출 플랫폼**으로 발전할 수 있습니다.
