# 완전 자동화 설정 가이드

## 🎯 목표: "컴퓨터만 켜져있으면 자동으로 작동"

---

## 방법 1: Windows 작업 스케줄러 (로컬 PC)

### 설정 방법

#### 1. 뉴스 수집 자동화 (매일 새벽 2시)

1. 작업 스케줄러 열기
   - `Win + R` → `taskschd.msc`

2. 기본 작업 만들기
   - 이름: "PsyInsight 뉴스 수집"
   - 트리거: 매일, 새벽 2시
   - 작업: 프로그램 시작
   - 프로그램: `python`
   - 인수: `C:\EX\collect_news.py`
   - 시작 위치: `C:\EX`
   - **"사용자가 로그온할 때만 실행" 체크 해제**
   - **"가장 높은 수준의 권한으로 실행" 체크**

3. 완료!
   - PC만 켜져있으면 매일 자동 실행
   - 프로그램 실행 불필요

#### 2. 논문 수집 자동화 (매주 월요일 새벽 2시)

동일한 방법으로 설정:
- 이름: "PsyInsight 논문 수집"
- 트리거: 매주, 월요일, 새벽 2시
- 인수: `C:\EX\collect_papers.py`

## 방법 2: 백그라운드 서비스 (더 안정적)

### Python 스크립트를 Windows 서비스로 등록

1. `pywin32` 설치
   ```bash
   pip install pywin32
   ```

2. 서비스 스크립트 생성 (선택사항)
   - 더 복잡하지만 더 안정적
   - PC 재부팅 시 자동 시작

---

## 방법 3: 클라우드 배포 (완전 자동화, 추천!)

### Streamlit Cloud + GitHub Actions

#### 장점
- ✅ PC 꺼져도 작동
- ✅ 24/7 자동 실행
- ✅ 완전 무료
- ✅ 자동 도메인 제공

#### 설정 방법

1. **GitHub에 코드 업로드**
   ```bash
   git add .
   git commit -m "배포 준비"
   git push
   ```

2. **Streamlit Cloud 배포**
   - https://streamlit.io/cloud 접속
   - GitHub 로그인
   - "New app" → 저장소 선택
   - 배포 완료!

3. **GitHub Actions로 자동 수집 설정**
   - `.github/workflows/collect_news.yml` 생성
   - 매일 자동 실행 설정
   - 수집 후 이메일 발송

## 🔧 현재 작동 방식

### 현재
1. `streamlit run app.py` 실행 → 웹 브라우저에서 접속
2. 수동으로 "뉴스 수집" 버튼 클릭
3. 또는 `python collect_news.py` 수동 실행

### 완전 자동화 후
1. PC만 켜져있으면 자동 실행
2. 매일 새벽 2시 자동 수집
3. 수집 완료 후 자동 이메일 발송
4. 데이터베이스에 자동 저장
5. 필요할 때 웹에서 확인
