## 다음 단계

### 1. GitHub 저장소 생성 (필요한 경우)

GitHub에 저장소가 없다면:
1. https://github.com/new 접속
2. 저장소 이름: `psyinsight-commander` (또는 원하는 이름)
3. Public 또는 Private 선택
4. "Create repository" 클릭

### 2. GitHub에 푸시

저장소가 생성되면:
```bash
git remote add origin https://github.com/당신의계정/psyinsight-commander.git
git branch -M main
git push -u origin main
```

**인증 필요:**
- GitHub Personal Access Token 사용 권장
- 또는 GitHub Desktop 사용

### 3. Streamlit Cloud 배포

1. https://streamlit.io/cloud 접속
2. GitHub로 로그인
3. "New app" 클릭
4. 저장소 선택
5. Main file: `app.py`
6. "Deploy!" 클릭
