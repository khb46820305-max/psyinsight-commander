# 프로젝트 변경 사항 요약

## 📝 변경 내역

### Tab 2 (소셜 마인드 앤 이모션 리스너) 제외 결정

**이유:**
- 국내 커뮤니티(네이트판/블라인드) API 접근 어려움
- 스크래핑 시 차단 위험 및 로그인 필요
- 사용자가 많은 국내 사이트가 아니면 효용성 낮음
- 차단당하지 않으면서 사용 가능한 방법이 없음

**결과:**
- Tab 2 완전 제외
- Tab 3 → Tab 2로 재번호
- Tab 4 → Tab 3로 재번호

## 🎯 최종 프로젝트 구조

### Tab 1: 사이콜로지 트랜드 레이더
- 미국/한국 심리 관련 뉴스 수집
- AI 요약 및 전문성 평가
- 검색, 필터, 정렬 기능

### Tab 2: 아카데믹 아카이브 (기존 Tab 3)
- 심리학 논문 수집 (arXiv, PubMed)
- AI 요약 (연구 목적, 방법, 결과, 시사점)
- 검색 및 필터 기능

### Tab 3: 콘텐츠 팩토리 (기존 Tab 4)
- Tab 1~2에서 선택한 콘텐츠로 재생산
- 블로그 포스트, 릴스 대본, 게시글, 논문 아이디어 생성

## 📊 작업 시간 재계산

### 기존 예상 시간
- MVP: 30-40시간
- 전체: 70-100시간

### 변경 후 예상 시간
- **MVP: 30-40시간** (변경 없음)
- **전체: 50-70시간** (Tab 2 제외로 20-30시간 단축)

## 🌐 프로젝트 형태 명확화

### 웹 애플리케이션 (홈페이지/웹사이트)
- **기술**: Streamlit 기반 대시보드
- **접근**: 웹 브라우저에서 접속
- **실행 방식**:
  - 로컬: `streamlit run app.py` (개인 PC)
  - 클라우드: Streamlit Cloud 배포 (인터넷 접속)
- **사용자 경험**: 일반 웹사이트처럼 탭 클릭하여 기능 사용
