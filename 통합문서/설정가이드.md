## 사용 가이드

### 1. 초기 설정

#### 의존성 설치
```bash
pip install -r requirements.txt
```

#### 환경 변수 설정 (.env 파일)
```env
GEMINI_API_KEY=your_api_key
EMAIL_SENDER=your-email@gmail.com
EMAIL_PASSWORD=your_app_password
EMAIL_RECEIVER=receiver@example.com
```

#### 데이터베이스 초기화
```bash
python -m modules.database
```

### 2. 앱 실행
```bash
streamlit run app.py
```

## 이메일 설정

### Gmail 앱 비밀번호 생성
1. Google 계정 설정 > 보안
2. 2단계 인증 활성화
3. 앱 비밀번호 생성
4. 생성된 비밀번호를 `.env` 파일의 `EMAIL_PASSWORD`에 입력

## 자동화 설정

### Windows Task Scheduler
1. 작업 스케줄러 열기
2. 기본 작업 만들기
3. 트리거: 매일 원하는 시간
4. 동작: 프로그램 시작
   - 프로그램: `python`
   - 인수: `C:\경로\collect_news.py`
5. 저장

### 자동화 스크립트
- `collect_news.py`: 뉴스 수집
- `collect_papers.py`: 논문 수집

## Streamlit이란?

Streamlit은 Python으로 웹 앱을 쉽게 만들 수 있는 프레임워크입니다.

### 특징
- Python 코드만으로 웹 앱 제작
- 실시간 업데이트
- 간단한 문법

### 예시 사이트
- https://streamlit.io/gallery

---

## 자동화 작동 방식

1. **스케줄러 실행**: 매일 지정 시간에 스크립트 실행
2. **데이터 수집**: 뉴스/논문 자동 수집
3. **AI 분석**: 자동 요약 및 평가
4. **이메일 발송**: 수집된 헤드라인 이메일 발송
5. **데이터베이스 저장**: 수집된 데이터 저장
